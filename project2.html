VOCAP – Vocabulary Companion

VOCAP – Vocabulary Companion is a personal vocabulary learning and tracking Android application designed to help users improve their English vocabulary in a structured and motivating way. The application works like a personal vocabulary notebook where users can store new words, review them later, and track their vocabulary growth over time. Unlike many vocabulary apps that rely heavily on internet connectivity, VOCAP focuses on offline learning. Words are stored locally using an SQLite database which allows the application to work smoothly even without internet access. The goal of this project is to provide a simple but powerful tool that encourages daily vocabulary learning and helps users build a long-term learning habit.

Key Features

  • Add new vocabulary words with meaning, category and example sentence.
  • View all saved words in a clean and organized word list.
  • Edit or update existing words anytime.
  • Delete words that are no longer needed.
  • Daily "Word of the Day" feature to encourage consistent learning.
  • Total words counter showing vocabulary growth.
  • Category statistics showing nouns, verbs and adjectives.
  • Learning streak tracker to motivate daily study.
  • Vocabulary growth chart showing progress over time.
  • Category distribution chart using a pie or donut chart.
  • Word pronunciation support using audio playback.
  • Export vocabulary list as a downloadable PDF.
  • Share the application easily with friends.
  • Privacy policy and rating option for user trust.

App Screenshots

Technical Implementation

VOCAP is developed as a fully offline Android application using Java and Android Studio. The vocabulary words are stored locally using an SQLite database which ensures fast performance and offline accessibility. The home screen displays the vocabulary list using RecyclerView. Users can add new words through a dedicated form interface. The application also uses chart libraries to visualize vocabulary growth and category statistics.

Technology Stack

  • Java
  • Android Studio
  • SQLite Database
  • RecyclerView
  • Chart Library for statistics
